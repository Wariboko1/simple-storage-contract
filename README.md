# Simple Storage Smart Contract

## About
My first Solidity smart contract built while studying 
Patrick Collins' Smart Contract Engineering course.

## What This Contract Does
- Stores a favourite number on the blockchain
- Retrieves the stored number
- Stores a list of people with their favourite numbers
- Maps names to favourite numbers

## Technologies Used
- Solidity ^0.8.18
- Ethereum Virtual Machine (EVM)
